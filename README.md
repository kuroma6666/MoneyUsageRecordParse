# SonyBankUsageRecordParse

## 概要

- 利用明細CSVから費用項目毎の使用金額を算出するGUIアプリ

## S/W構成

| 名称          | バージョン | 備考 |
| :------------ | ---------- | ---- |
| Visual Studio | 2022       |      |
| .NET          | 8.0        |      |

## 動作対象OS

- Windows10、Windows11が対象となります。
- Windows10以前については動作を保証しません。

## サポートしている利用明細CSV

| 名称           | サポート | 備考 |
| :------------- | -------- | ---- |
| SonyBankWallet | ✅       |      |
| SMBC Card      | ✅       |      |
